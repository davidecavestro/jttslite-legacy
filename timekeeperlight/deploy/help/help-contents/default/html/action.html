Action

An action fixes a temporal space during which the user devote himself to a certain task.

Therefore it comes with a term, calculated as the difference between the end time and start time. It also has a note field.

When an action has a start time but lacks of the end one, it is called the current action.

The actions of the currently selected task are listed on the actions table.

You have several ways to create an action:

  • starting an action

  • adding an action

  • starting an action clone

Actions can also be edited, deleted; you can cut/copy and paste them like every other object.

When you do several times the same job, with the same description or duration, you can add it to the action templates list, and use it as a basis for the creation of new actions.



Starting an action

Use the menu |Actions |Start action| to create a current action, or rather a new action, with the present time as start time, but without an end time. You cannot have more than one current action at a time.

This operation is useful to record your effort while you are working, and automatically measure its duration.

Adding an action (stopped)

From the |Actions|Add action| menu you can launch the “Add action” dialog to create a new action setting a start and end time.

This operation is useful to record your effort when the work is finished.

Of course, you have to know the start and end time.


Illustration 1: The "Add action" dialog




Choose a template if you want to reuse the description and duration from an action template, or, as usual, simply put a start and end time or a duration.

Use the tool-button on the right of the action template drop-down list as a shortcut to edit the available action templates.

Starting an action clone

Using the menu item |Actions|Start action clone| you obtain the same result as starting an action, with the difference that the new action has the same notes of the selected one.

Stopping the current action

To stop the current action, choose the menu item |Actions|Stop action|.

Deleting an action

To delete an action, select it and choose the menu item |Edit|Delete|.

You cannot delete the current action.

Continue an action

To continue a previously stopped action, as it was never stopped, choose the menu item |Actions|Continue action|.

Add an action to templates

To add an action to the templates list, select the action, right-click on it, and choose Add to templates from the contextual menu. A new entry will be available in the action template editor.
