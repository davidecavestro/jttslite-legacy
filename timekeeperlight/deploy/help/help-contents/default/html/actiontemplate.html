Action template

A template is an object that holds a description and a duration that can be used to “derive” an action.

It is useful when you have to do the same job again and again, having always the same description and maybe a similar duration; in that case, you can create a template, set it the pertinent data, and then create actions from that template.

To use a template, simply open the “Add action” dialog, and choose a template.

To add, remove or edit your templates open the action templates editor selecting |Tools|Action Templates| from the main menu.
