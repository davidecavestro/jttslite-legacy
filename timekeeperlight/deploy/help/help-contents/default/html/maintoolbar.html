The toolbar

The main toolbar offers you a fast way to do some useful operations.

However, every operation remains available even from the menu on the top of the window.


Illustration 1: the toolbar




From left to right, you will find the following buttons:

New workspace

Create a new workspace


New task

Create a new task


Open workspace

Open an existing workspace


Add action

Add a new action, with a start and end time


Start action

Start a new action and leave it open, as the current one


Stop action

Stop the current action


Cut

Cut the selected tasks or actions


Copy

Copy the selected tasks or actions


Paste

Paste the tasks or actions previously cut or copied


Undo

Undo the last change


Redo

Redo the last change


Edit templates

Open the action template editor to update you templates collection.


Reporting

Launch the reports panel


Show help

Show the application help system
