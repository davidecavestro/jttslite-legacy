Task

The task is a job for which you want to record you spend on it. It can be decomposed in several sub-tasks.

Every workspace comes with only one task (its main task) and by default it has the same name of the workspace.

The main task is the root of the task tree; the first level of the tree is made up of the root sub-tasks, and so on, to form a hierarchical structure. There aren't limitations on the number of available levels.

Every task tree element can have some actions that make its total effort .

For every task you are available the following operations:

  • add a task

  • delete a task

  • rename a task

  • move a task

You can also cut, copy and paste every task.

When you copy a task, will be copied recursively every subtask, with its respective actions, excluded the current one.



Add a task

To create a new task under an existing one (so adding a sub-task to it), first select the task and choose |File|New task| from menu.

The new sub-task will be added after the existing sub-tasks, with a predefined name derived from its parent; but you can rename it later.

Delete a task

To delete a task choose |Edit|Delete| from the menu.

Warning: when you delete a task, even all its sub-tasks and actions will be deleted.

Rename a task

To rename a task, select it within the task tree, click with the right mouse button and choose the Rename contextual menu item.

Move a task

Sometimes it can be useful to move a task, to follow some logical ordering of your activities, or simply to reorganize the tree according to a different criteria.

To move a task, select it within the tree and choose |Edit|Cut| from menu.

Then select a target task and choose |Edit|Paste|.

You will have to choose, from a dialog, if you want to:

  • paste before the the selected task

  • paste after the selected task

  • paste as last sub-task of the selected one.
