Workspace

A workspace is a task container.

You can organize your tasks grouping them by workspace, to reduce the number of task contemporary available.

Every workspace has a task tree, and every task in the tree can have some actions associated.

These are the main operations available on a workspace:

  • create a workspace

  • open a workspace

  • export a workspace

  • import a workspace



Create a workspace

The application comes with a predefined workspace. You can create new workspaces choosing |File|New workspace| from the menu.

Open a workspace

You cannot use more than one project simultaneously. To change workspace, choose |File|Open workspace|.

Export a workspace

You can export current workspace's data (including task tree and its actions) choosing |File|Export workspace|. The export utility could be useful, for example, to make backup copies of your data, to transfer the data from a computer to another, etc.

Import a workspace

To import a workspace, choose |File|Import workspace|.

If you are importing data from a workspace having the same name of an existing one, you will be asked to choose if you want to overwrite, import with a different name, or cancel the operation.

Obviously, in case you are restoring a backup, you can overwrite the workspace.

Then, to access the imported workspace you have to open it.
