Progetto

Il progetto è un “contenitore” di attività.

E' possibile organizzare le attività raggruppandole per progetto, in modo da ridurre il numero delle attività contemporaneamente disponibili.

Ogni progetto ha un albero delle attività associato, ogni attività può avere delle azioni.

Le principali operazioni disponibili per il progetto sono:

  • creare un progetto

  • aprire un progetto

  • esportare un progetto

  • importare un progetto



Creare un progetto

L'applicazione viene fornita con un progetto predefinito. E' possibile creare altri progetti dal menù |File|Nuovo progetto|.

Aprire un progetto

L'applicazione può utilizzare un solo progetto per volta. Per cambiare progetto corrente utilizzare la voce |File|Apri progetto| dal menù principale.

Esportare un progetto

E' possibile esportare in dati del progetto corrente (compresi albero delle attività e relative azioni) dal menù |File|Esporta progetto|. L'esportazione può risultare utile, ad esempio, per effettuare dei backup dei dati, per trasferire i dati da un computer ad un altro, etc.

Importare un progetto

Per importare un progetto selezionare il menù |File|Importa progetto|.

Nel caso in cui si stia importando un progetto con il medesimo nome di uno esistente, verrà chiesto di scegliere se sovrascrivere, importare con un nome diverso o annullare.

Nel caso di ripristino di un file di backup si può sovrascrivere.
